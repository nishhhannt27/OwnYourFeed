Privacy Policy for OwnYourFeed

OwnYourFeed does not collect, store, or transmit any personal user data.

Data Storage
The extension stores user preferences locally using the Chrome storage API. This includes settings such as whether focus mode is enabled and selected topic categories. This information remains only on the user's device and is never transmitted to external servers.

Data Collection
OwnYourFeed does not collect any personally identifiable information, including names, email addresses, or account information.

Third-Party Sharing
OwnYourFeed does not sell, transfer, or share any user data with third parties.

Website Content Access
The extension interacts with YouTube pages only to modify the interface and display a curated feed based on user-selected topics. This processing happens locally in the browser.

Contact
If you have questions about this privacy policy, please contact the developer through the GitHub repository.
